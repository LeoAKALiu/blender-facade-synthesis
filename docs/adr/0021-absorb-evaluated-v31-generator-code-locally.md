# Absorb evaluated V3.1 generator code locally

This project will first assess the usable modules in the specified `img2height`
V3.1 synthetic-facade generator, then move the selected code into this
repository under local ownership. The Web Studio will not depend on a git
submodule, remote package, or ongoing upstream synchronisation; all subsequent
development happens in this project.

## Considered Options

- A submodule or runtime dependency preserves a connection to the old project
  but couples a new dataset application to its upstream history and releases.
