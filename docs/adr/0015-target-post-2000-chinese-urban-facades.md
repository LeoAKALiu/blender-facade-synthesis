# Target post-2000 Chinese urban facades

The first-release scene grammar, materials, window layouts, podiums, and
contextual elements will approximate Chinese urban buildings constructed or
materially renovated from the 2000s onward. This target-domain field is part of
the recipe and package provenance, not an unstated visual preference.

## Considered Options

- A global, style-agnostic facade library would appear diverse but dilute the
  geometry and material cues needed for the intended real-image domain.
